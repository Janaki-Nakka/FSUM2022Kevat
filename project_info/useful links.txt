Useful links to HTML studies
============================

This web site describes history and background of various HTML versions:
https://en.wikipedia.org/wiki/HTML

This is a good place to learn basics. It has interactive tutorials:
https://www.w3schools.com/html/default.asp

MDN is excellent place to learn HTML
https://developer.mozilla.org/en-US/docs/Learn/HTML
https://developer.mozilla.org/en-US/docs/Web/HTML

Tutorialspoint is alternative place to learn HTML
https://www.tutorialspoint.com/html/index.htm
https://www.tutorialspoint.com/html5/index.htm

Very good material for those writing professionally HTML and CSS:
https://internetingishard.com/

HTML5 Microdata
https://splatworld.tv/what-is-microdata-a-detailed-look-at-schema-org/

Semantic HTML theory
https://internetingishard.com/html-and-css/semantic-html/

https://www.w3schools.com/js/default.asp

http://javascript.info/
https://exploringjs.com/
https://2ality.com

Tämän viikon aikana kannattaa käydä läpi seuraavat verkkotehtävät (ja seuraavassa järjestyksessä):
